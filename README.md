# Spam Email Classifier

A Machine Learning project built using Python that detects whether a message is **Spam** or **Not Spam** using **Natural Language Processing (NLP)** and the **Naive Bayes Algorithm**.

This project uses text vectorization techniques to convert messages into numerical data and trains a machine learning model to identify spam messages with high accuracy.

## Features
- Spam and Ham message classification
- Text preprocessing and vectorization
- Machine Learning based prediction
- High accuracy spam detection
- Beginner-friendly AI project

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning Concepts
- Natural Language Processing (NLP)
- CountVectorizer
- Naive Bayes Algorithm
- Train-Test Split
- Text Classification

## Dataset
SMS Spam Collection Dataset

## How to Run

Install required libraries:

```bash
pip install pandas numpy scikit-learn
```

Run the project:

```bash
python spam_classifier.py
```
